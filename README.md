# SaraArai

Unityで制作中の、皿洗いを題材にした2Dゲームです。

## 概要

汚れた皿をマウスでこすって洗い、きれいになった皿を売却やスポンジの強化、今後追加予定のアクション要素につなげるゲームを目指しています。

## 操作方法

- 皿を出すボタン: 汚れた皿をテーブルに出す
- テーブル上の皿をクリック: 洗い画面を開く
- テーブル上の皿をドラッグ: 皿の位置を移動する
- テーブルの右上のお金が書かれた四角に皿をドロップ: お金が手に入る
- テーブルの右下のStockと書かれた四角に皿をドロップ: 弾幕用の皿が貯まる
- 洗い画面で左クリックしながらマウス移動: 汚れを落とす
- 洗い皿の外をクリック: 洗い画面を閉じる

## こだわった点

皿洗いを単なる作業で終わらせず、洗った皿を売ってお金にするか、今後追加予定のアクションパート用に貯めるかを選べるゲームループにすることを意識しました。洗う、売る、強化する、使うという流れを作ることで、皿を洗う行為自体に次の目的が生まれるように設計しています。

## 今後追加したい要素

- お金によるスポンジ強化
- 皿を洗い終えた時のエフェクト
- 洗った皿を使う弾幕・アクションパート
- 弾幕で使える皿と洗ってストックした皿の同期
